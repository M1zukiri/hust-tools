# Location — 模糊定位引擎

通过多个关键词搜索高德地图 POI，利用密度极大值算法实现模糊定位。

## 调用方法

```bash
python main.py
```

> 程序会读取代码中预设的关键词列表，调用高德地图 API 进行搜索，\
> 然后通过权重算法推测目标位置。

## 使用步骤

1. **申请高德地图 API Key**（Web 服务）\
   访问 https://lbs.amap.com/ → 控制台 → 创建应用 → 获取 Key

2. **在代码中填入 Key**\
   打开 `main.py`，找到第 252 行，填入你的 Key：
   ```python
   AMAP_KEY = "你的高德Web服务Key"
   ```

3. **修改搜索关键词**\
   找到第 265 行的 `user_inputs` 列表，替换为你想定位的关键词：
   ```python
   user_inputs = ["东方明珠", "陆家嘴", "上海中心大厦"]
   ```

4. **运行**\
   ```bash
   python main.py
   ```

## 输出结果

- 定位描述（最近地点名称）
- 中心坐标（GCJ-02 坐标系）
- 置信度得分
- 匹配到的关键词列表
- 高德地图链接（可打开查看位置）

## 核心算法

1. **数据获取** — 调用高德地图 POI 搜索 API 获取每个关键词的地理坐标
2. **权重计算** — 根据关键词返回的 POI 数量计算稀缺性权重（越少见的词权重越高）
3. **密度极大值搜索** — 遍历每个 POI 作为圆心，统计半径内加权和，取最高分作为定位结果
