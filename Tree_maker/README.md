# Tree_maker — 目录树生成器

递归生成指定目录的树状结构图，并自动保存为带时间戳的文本文件。

## 调用方法

```bash
# 在当前目录生成完整目录树（不限深度）
python tree_maker.py

# 限制深度为 2 层
python tree_maker.py -d 2

# 指定更多层
python tree_maker.py -d 5
```

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `-d, --depth` | int | -1（不限） | 遍历深度层级 |

## 输出文件

自动生成文件名为 `<目录名>_depth-<深度>_<时间戳>.txt`，例如：

```
项目A_depth-all_20231027_153000.txt
项目A_depth-2_20231027_153001.txt
```

## 项目结构

```
Tree_maker/
├── tree_maker.py    # 主程序（生成目录树）
└── README.md        # 本文档
```

## 特性

- 文件夹排在前，文件排在后，按字母序排列
- 使用 `├──` / `└──` / `│` 等树形符号，生成美观的树状图
- 遇到权限不足的目录自动跳过，不会崩溃
- 文件名包含深度和生成时间，便于追溯

## 依赖

无（仅使用 Python 标准库 `argparse`、`pathlib`、`datetime`）
